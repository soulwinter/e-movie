# 登录注册功能

## 用户登录接口（电话+密码）

#### 接口地址

`POST /user/loginWithPassword`

#### 请求参数

请求参数应该包括以下信息：

| 参数名    | 类型   | 是否必须 | 默认值 | 描述     |
| --------- | ------ | -------- | ------ | -------- |
| telephone | string | 否       | 无     | 手机号码 |
| password  | string | 否       | 无     | 密码     |

#### 请求示例

```
{ 
  "telephone": 18312345678,
  "password": "example_password", 
}

```


#### 响应参数

响应参数应该包括以下信息：

| 参数名   | 类型   | 描述       |
| -------- | ------ | ---------- |
| success  | int    | 状态码     |
| errorMsg | string | 响应信息   |
| data     | object | 返回的数据 |
| total    | int    | 数据数量   |

#### 响应示例

```json
#登陆成功
{ 
    "success": true, 
    "errorMsg": null ,
    "data": {
    		"token":"723975983745893475893hiuerh893y4899"
		"username":"是刘铎啊~"
		},
    "total":null
}

#登录失败
{ 
    "success": false, 
    "errorMsg": "手机号格式错误，不合法" ,
    "data": null,
    "total":null
}
    
```

#### 补充说明

```
登录失败返回的"errorMsg"有一下字段：
	"手机号不能为空"
	"密码不能为空"
	"手机号格式错误，不合法"
	"该手机号还未注册或密码有误"
```













## 获取验证码接口

#### 接口地址

`POST /user/code`

#### 请求参数

请求参数应该包括以下信息：

| 参数名    | 类型   | 是否必须 | 默认值 | 描述                   |
| --------- | ------ | -------- | ------ | ---------------------- |
| telephone | string | 是       | 无     | 手机号                 |
| mode      | string | 是       | 无     | 是登录模式还是注册模式 |

#### 请求示例

```json
{ 
    "telephone":"18866667777"
    "mode":"login_mode"
}

{ 
    "telephone":"18866667777"
    "mode":"register_mode"
}
```


#### 响应参数

响应参数应该包括以下信息：

| 参数名   | 类型   | 描述     |
| -------- | ------ | -------- |
| success  | int    | 状态码   |
| errorMsg | string | 响应信息 |

#### 响应示例

```json
#成功
{ 
    "success": true, 
    "errorMsg": null ,
}

#登录失败
{ 
    "success": false, 
    "errorMsg": "手机号格式错误，不合法" ,
}
    
```
